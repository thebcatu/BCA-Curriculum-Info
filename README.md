## Overview

The BCA program is a four-year degree comprising 126 credit hours, equivalent to 2016 lecture hours. The curriculum is designed to provide comprehensive knowledge in computer applications with a mix of core courses, electives, mathematics, languages, and social science subjects.

## How to Use

- **PDF Version**: [Download the complete syllabus in PDF format](https://example.com/Syllabus-BCA-TU.pdf)

## Data Structure

The curriculum data is organized in the following structure:
- Markdown tables for easy reading directly on GitHub
- CSV files for importing into spreadsheet applications
- JSON data for programmatic access
- PDF documents for official syllabus references
